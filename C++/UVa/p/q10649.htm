Q10649: Danger Point

Jamal 和 Kamal是朋友，而且在他們住的區域他們的友誼為人所樂道。有一次Jamal向Kamal借了一大筆錢。起初Kamal並不太介意，可是到了期限Jamal卻始終不還錢。當Kamal瞭解到Jamal不打算還他錢之後，他就對Jamal非常不客氣。結局是他們現在像仇人一樣。

三個月之後，搬來了另外2個人，Rahim和Karim。很奇特的，他們的房子座落的位置呈現如下的特性：

  • Rahim連接Jamal的直線和Rahim連接Kamal的直線恰好是垂直的。同樣的情形也發生在Karim的房子。
  • 連接Rahim和Karim的直線和連接Jamal和Kamal的直線相交夾角為45度。交點被稱為「危險的交叉點（Danger Point）」。

如果你還是不清楚的話請參考下圖：

你知道為什麼稱為「危險的交叉點」嗎？因為Rahim和Karim也有嚴重的金錢糾紛。所以當他們4個人在「危險的交叉點」碰頭時，就會發生激烈的衝突。這件事困擾大家許久，要解決這個問題，首先得解決一個小問題：找出Rahim的房子到「危險的交叉點」的距離。

Input

每組測試資料一列，含有 2 個數 。第一個數代表Jamal和Kamal房子之間的距離。第二個數代表「危險的交叉點」到 Karim 房子的距離。

若這2個數均為 0 代表輸入結束。請參考Sample Input。

Output

對每組測試資料輸出一列，「危險的交叉點」到 Rahim 房子的距離。請輸出到小數點後6位。

如果輸入的資料不正確，則輸出 INCORRECT INFORMATION !!!

Sample Input Sample Output
333.25 101.154
50.50 10.10
10.10 50.50
0 0
