Q10706: Number Sequence

給你一個正整數 i ，請你寫一個程式找出在數列 S1S2S3...Sk 中第 i 個位置的字元是多少。其中 Sk 包含從 1 到  k 的數字，一個接著一個。例如 S3=123，S11=1234567891011。

S1S2S3...的前 80 個字元如下所示：

11212312341234512345612345671234567812345678912345678910123456789101112345678910

Input

輸入的第一列有一個整數 t（ 1 <= t <= 25），代表以下有多少組測試資料。

每組測試資料一列。含有1個整數 i（ 1 <= i <= 2147483647）。請參考Sample Input。

Output

對每組測試資料輸出一列，輸出在數列 S1S2S3...Sk 中第 i 個位置的字元是多少。
