Q10693: Traffic Volume

在上圖中的馬路上有無限多輛的汽車在上面。連續2輛車之間的距離是 d ，每輛車的長度都是 L ，並且每輛車的速度都是 v 。車流量代表在一段時間內通過馬路的汽車的數目。當速度固定時，d 要最小才能使車流量最大。在這個情況下 d 最小的可能為 v2 / (2f)。（車子的速度越快，需要的煞車距離也就越長。）在這裡，f 是煞車的加速度。

在這個模型下，給你 L 和 f ，你的任務是找出 v 的值，使得通過馬路的車流量最大。

Input

輸入含有多組測試資料。

每組測試資料一列，含有 2 個整數 L（0 <  L <= 100）和 f（0 < f <= 10000）。L 的單位為 公尺，f 的單位為 公尺/秒2。

當 L = f = 0 時代表輸入結束，請參考Sample Input。

Output

對每組測試資料輸出一列，含有2個浮點數 。第一個數代表車子的速度 v ，可以使得通過馬路的車流量最大。第二個數為一個小時內通過馬路的車輛最大數目（當然，這可以含有小數的）。這2個浮點數請輸出到小數點後 8 位。小於 1e-5 的誤差將會被忽略。請參考 Sample Output。

Sample Input Sample Output 
5 3
0 0

5.47722558 1971.80120702
